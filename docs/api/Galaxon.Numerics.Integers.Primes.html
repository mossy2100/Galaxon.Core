Eratosthenes(UInt32)

Use the Sieve of Eratosthenes to update the cache for all prime numbers less than or equal to a given maximum. https://en.wikipedia.org/wiki/Sieve_of_Eratosthenes

I've limited the value of max to uint.MaxValue (4,294,967,295) as I think this will be adequate for practical purposes, and it means the isComposite cache will not need to be larger than Array.MaxSize. Nor will we need to use batches or multiple arrays for isComposite.
