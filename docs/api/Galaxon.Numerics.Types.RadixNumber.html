Class RadixNumber

This class supports numbers with a specified radix, which can be in the range 2..36.

Additional support is provided in the form of "To" and "From" methods for:

  • binary (base 2)
  • quaternary (base 4)
  • octal (base 8)
  • decimal (base 10)
  • duodecimal (base 12)
  • hexadecimal (base 16)
  • vigesimal (base 20)
  • triacontakaidecimal (base 32)
