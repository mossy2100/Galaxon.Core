Sqrt(DecimalComplex)

Calculate the square root of a DecimalComplex number. The second root can be found by the negative of the result, as with other Sqrt() methods. You can use this method to get the square root of a negative decimal. e.g. DecimalComplex z = DecimalComplex.Sqrt(-5); (There will be an implicit cast of the -5 to a DecimalComplex.) TODO Test.
